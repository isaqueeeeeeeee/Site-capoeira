O QUE EU CRIEI:
Desenvolvi um site institucional responsivo para nosso grupo de capoeira, com foco em apresentar nossa história, professores, aulas e facilitar o contato com interessados.
ESTRUTURA DO SITE:
1. PÁGINA PRINCIPAL (index.html)
Cabeçalho fixo com menu de navegação

Seção Hero com animação de entrada

Perfil do Mestre Jiboia (nosso fundador)

Carrossel automático dos professores

Informações sobre o grupo

Grade de aulas em diferentes locais

Formulário de contato funcional

Rodapé com redes sociais

 2. PÁGINA DE PROFESSORES (professores.html)
Página dedicada com fotos e descrições

Linkada ao menu principal

TECNOLOGIAS QUE USEI:
HTML5 semântico
CSS3 com variáveis (custom properties)
Flexbox e Grid para layout
Animações CSS (keyframes)
Design responsivo (mobile-first)
Formulário integrado ao Formspree
Fontes do Google Fonts

FUNCIONALIDADES IMPLEMENTADAS:
Funcionando perfeitamente:
Navegação suave entre seções

Formulário de contato que envia para meu email

Links de redes sociais (Instagram, Facebook, YouTube)

Design responsivo para todos os dispositivos

Carrossel automático de professores

Animações visuais (hover, fade-in)
